# 📊 Smart Business Analytics & Prediction System

## 🚀 Overview
An AI-powered business analytics platform that allows users to upload datasets, visualize insights, predict future trends, detect anomalies, and generate reports.

## 🔥 Features
- 📊 Interactive Dashboard (Plotly)
- 🤖 Machine Learning Prediction (Random Forest)
- 🔍 Feature Importance Analysis
- 📈 Sales Forecasting
- 🚨 Anomaly Detection
- 📄 PDF Report Generation
- 🤖 AI-based Insights
- 🎛️ Filters & KPIs
- 📊 Model Comparison

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- Plotly
- ReportLab

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
